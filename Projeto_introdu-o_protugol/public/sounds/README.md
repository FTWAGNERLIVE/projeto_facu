# Sons do Jogo

Esta pasta contém os arquivos de áudio do jogo.

## Estrutura de Sons

Adicione os seguintes arquivos de som nesta pasta:

### Efeitos Sonoros (Opcionais)
- `move.mp3` - Som quando o personagem se move
- `key-collect.mp3` - Som ao coletar a chave
- `goal-reach.mp3` - Som ao chegar no destino
- `collision.mp3` - Som ao colidir com parede
- `button-click.mp3` - Som ao selecionar um comando ou clicar em botões

### Música de Fundo (Opcional)
- `background-music.mp3` - Música de fundo do jogo

## Formato Recomendado

- **Formato**: MP3 ou OGG (MP3 tem melhor compatibilidade)
- **Qualidade**: 128-192 kbps (balance entre qualidade e tamanho)
- **Duração dos efeitos**: 0.5-2 segundos
- **Música de fundo**: Pode ser mais longa (será repetida em loop)

## Onde Encontrar Sons Gratuitos

- **Freesound.org** - Biblioteca gratuita de sons
- **Zapsplat.com** - Efeitos sonoros gratuitos
- **OpenGameArt.org** - Recursos de áudio para jogos
- **Incompetech.com** - Músicas gratuitas (Kevin MacLeod)

## Nota

Se algum arquivo não existir, o jogo continuará funcionando normalmente, apenas sem o som correspondente.
