===============================================
        COMO FUNCIONA O SISTEMA
===============================================

1. ARQUIVO INDIVIDUAL (por participante):
   - Nome: quiz_result_[NOME]_[TIMESTAMP].txt
   - Contém: resultado detalhado com todas as respostas
   - Baixado automaticamente após completar o quiz

2. ARQUIVO DE REGISTRO GERAL:
   - Nome: REGISTRO_QUIZ_ATUALIZADO.txt
   - Contém: lista simples com todos os participantes
   - Baixado automaticamente após cada quiz

3. RANKING COMPLETO:
   - Nome: RANKING_QUIZ_[DATA].txt
   - Contém: ranking ordenado + estatísticas
   - Baixado manualmente clicando em "Exportar Ranking"

===============================================
        FUNCIONALIDADES IMPLEMENTADAS
===============================================

✅ Registro automático de cada participante
✅ Download automático de relatório individual
✅ Download automático de registro geral
✅ Exportação manual do ranking completo
✅ Estatísticas gerais (média, maior, menor pontuação)
✅ Detalhamento de todas as respostas
✅ Timestamp para controle temporal
✅ Formatação organizada e legível

===============================================
        COMO USAR
===============================================

1. O participante completa o quiz
2. Digite o nome quando solicitado
3. O sistema automaticamente:
   - Salva no localStorage (ranking na tela)
   - Baixa arquivo individual detalhado
   - Baixa arquivo de registro geral
4. Para exportar ranking completo:
   - Clique em "Exportar Ranking" na tela de resultados
