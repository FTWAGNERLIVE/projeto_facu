# Início Rápido do Projeto

## Como iniciar o projeto

Agora você pode iniciar o projeto de **3 formas diferentes**, todas funcionam:

### Opção 1: Usando npm no diretório raiz (RECOMENDADO)
```powershell
npm start
```
Agora funciona diretamente do diretório raiz!

### Opção 2: Usando script PowerShell
```powershell
.\start.ps1
```

### Opção 3: Usando script Batch
```cmd
start.bat
```

### Opção 4: Entrando no diretório do projeto
```powershell
cd Projeto_introdu-o_protugol
npm start
```

## Outros comandos úteis

Do diretório raiz, você também pode usar:
- `npm run build` - Para fazer build de produção
- `npm test` - Para executar testes
- `npm run install-deps` - Para instalar dependências

## Nota

O `package.json` no diretório raiz redireciona automaticamente os comandos para o projeto correto, então você não precisa mais se preocupar com o diretório!
